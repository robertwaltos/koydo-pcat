# koydo-pcat

PCAT exam preparation — powered by Koydo.

## Quick start

```bash
npm install
npm run dev
```

## Stack

- **Framework:** Next.js 16.2.0 + React 19
- **Styling:** Tailwind CSS 4
- **Backend:** Supabase (shared)
- **Deploy:** Vercel

## Exam Metadata

| Field | Value |
|-------|-------|
| Exam ID | EXAM169 |
| Name | PCAT |
| Country | United States |
| Category | Admissions |
| Tier | Tier 10 |
| Wave | Deprioritize |
